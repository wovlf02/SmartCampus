## 1. 프로젝트 소개

| 항목 | 내용 |
|------|------|
| **프로젝트명** | Smart Campus (교내 네비게이션 및 커뮤니티 앱) |
| **버전** | 0.0.1 |
| **대상 사용자** | 대학생, 외국인 유학생, 캠퍼스 방문자 |
| **서비스 형태** | 모바일 애플리케이션 (iOS / Android) |

**Smart Campus**는 대학 캠퍼스 내 사용자 경험을 혁신하는 통합 모바일 애플리케이션입니다. 실시간 캠퍼스 지도 및 경로 안내, 커뮤니티 기반 정보 공유, 다국어 지원 등의 기능을 제공합니다.

---

## 2. 개발 배경

### 2.1 현재 문제점
- **길찾기 어려움**: 넓은 캠퍼스에서 초행자의 건물/시설 위치 파악 곤란
- **정보 접근 불편**: 분산된 캠퍼스 정보 (건물 정보, 편의시설 등)
- **소통 채널 부재**: 재학생 간 소통 및 정보 공유 플랫폼 미흡
- **언어 장벽**: 외국인 유학생의 캠퍼스 생활 적응 어려움

### 2.2 개발 목표
- **실시간 경로 안내**: 캠퍼스 내 건물 간 최적 경로 제공
- **커뮤니티 플랫폼**: 게시판, 채팅을 통한 정보 공유 및 소통
- **다국어 지원**: i18n 기반 다국어 UI로 외국인 유학생 접근성 향상
- **시간표 기반 서비스**: 개인화된 시간표 관리 및 알림

## 3. 주요 기능

### 3.1 지도 및 네비게이션

| 기능 | 설명 |
|------|------|
| 캠퍼스 지도 | Kakao Map API 기반 실시간 캠퍼스 지도 표시 |
| 현재 위치 표시 | GPS 기반 사용자 현재 위치 마커 표시 |
| 건물 검색 | 캠퍼스 내 건물 및 시설 검색 |
| 경로 안내 | 현재 위치에서 목적지까지 경로 안내 |

### 3.2 커뮤니티

| 기능 | 설명 |
|------|------|
| 게시판 | 게시글 작성/수정/삭제, 이미지 첨부 |
| 댓글/대댓글 | 게시글에 댓글 및 대댓글 작성 |
| 좋아요 | 게시글 좋아요/좋아요 취소 |
| 신고 | 부적절한 게시글/댓글/사용자 신고 |
| 차단 | 특정 사용자 차단 |

### 3.3 채팅

| 기능 | 설명 |
|------|------|
| 1:1 채팅 | 다른 사용자와 1:1 실시간 채팅 |
| 그룹 채팅 | 여러 사용자 그룹 채팅방 |
| 파일 공유 | 이미지, 파일 전송 |
| 읽음 표시 | 메시지 읽음/안읽음 표시 |

### 3.4 친구 관리

| 기능 | 설명 |
|------|------|
| 친구 요청 | 다른 사용자에게 친구 요청 전송 |
| 친구 수락/거절 | 받은 친구 요청 처리 |
| 친구 목록 | 친구 목록 조회 및 관리 |
| 친구 차단 | 친구 차단/신고 |

### 3.5 시간표 관리

| 기능 | 설명 |
|------|------|
| 시간표 등록 | 개인 시간표 추가 |
| 시간표 조회 | 등록된 시간표 조회 |
| 시간표 수정/삭제 | 시간표 항목 수정 및 삭제 |

### 3.6 마이페이지

| 기능 | 설명 |
|------|------|
| 프로필 수정 | 닉네임, 프로필 이미지 변경 |
| 비밀번호 변경 | 비밀번호 수정 |
| 대학교 설정 | 소속 대학교 변경 |
| 언어 설정 | 앱 언어 변경 |
| 회원 탈퇴 | 계정 삭제 |

### 3.7 인증

| 기능 | 설명 |
|------|------|
| 회원가입 | 아이디, 비밀번호, 이메일, 대학교 등록 |
| 이메일 인증 | 이메일 인증코드 발송 및 검증 |
| 로그인 | 아이디/비밀번호 기반 로그인 |
| 소셜 로그인 | 카카오, 네이버, 구글, GitHub 로그인 |
| 비밀번호 찾기 | 이메일 기반 비밀번호 재설정 |
| JWT 토큰 | Access/Refresh Token 기반 인증 |
